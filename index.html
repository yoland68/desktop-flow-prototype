ComfyUI Desktop - Installation Flow Prototype

Navigate through the installation and onboarding screens:

Main Installation Flow

  • Step 1: Welcome / Double Click Install
  • Step 2a: Migration Detection (Optional)
  • Step 2b: Migration Configuration (Optional)
  • Step 3: Installation Directory & Model Selection
  • Step 4: Installation Progress (3 seconds)
  • Step 5: ComfyUI Main Window

Flow Paths:
• New Users: 1 → 2a (choose fresh install) → 3 → 4 → 5
• Existing Users: 1 → 2a (choose migrate) → 2b → 3 → 4 → 5

Key Features:
• Combined installation directory and model selection on one screen
• Fast 3-second installation (packages only)
• Models download in background after launch
